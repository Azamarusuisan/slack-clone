# Slackクローン（ダミーデータ版）

## 目的
Slack の全画面をダミーデータで構築し、2カラムレイアウト + レスポンシブの基礎を習得する。

## 技術スタック
- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- react-router-dom

## 画面構成
- **サイドバー**: 幅 260px / 背景色 Slack紫 `#611f69` / チャンネル 5 件を表示
- **ヘッダー**: 現在のチャンネル名を表示
- **メッセージ一覧**: ダミーメッセージ 10 件を表示
- **入力欄**: 画面下部に固定

## データ
- すべてダミーデータで構築する
- DB 接続・認証は **Stage 14 以降** で対応（現段階では未実装）

## レスポンシブ
- 375px 幅ではサイドバーをハンバーガーメニュー化する
